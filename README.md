## Introduction

Heelbot is a Ruby framework to build an awesome bot. We believe that every duplicated work can be done automatically.

## Installation

```
gem install heelbot
```

## Usage

```
# List All Bots
heelbot bot list

# Help of Bot
heelbot bot help BOT_NAME

# Run Bot
heelbot run BOT_NAME
```
